# Backend Inventory Control

## Dependencies
- Express
- Sequelize
- Sqlite3

## Install
```bash
$ npm install express
$ npm i sequelize
$ npm i sqlite3
$ npm install -g nodemon
```

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3005](http://localhost:3005) to view it in your browser.
